# PavanKumar N - Portfolio

A simple, clean, and professional portfolio website built with HTML and CSS, designed in the style of IIT Bombay academic pages.

## Features

- **Dark Theme**: Modern dark color scheme with blue accents
- **Profile Photo**: Circular profile image in the header
- **Clean Design**: Professional academic-style layout similar to IIT Bombay
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Print-Friendly**: Optimized for printing (converts to light theme)
- **Modern Typography**: Uses Google Fonts (Roboto) for better readability
- **Color-Coded Sections**: Different colors for different content types

## File Structure

```
├── index.html          # Main HTML file
├── styles.css          # CSS styling
├── profile-photo.jpg   # Profile photo (replace with actual image)
└── README.md           # This file
```

## How to Use

1. **Add Profile Photo**: Replace `profile-photo.jpg` with your actual photo (recommended size: 300x300px or larger)
2. **Open the Portfolio**: Simply open `index.html` in any modern web browser
3. **Local Development**: You can use any local server or just open the file directly
4. **Deploy**: Upload all files to any web hosting service

## Customization

### Personal Information
Edit the following sections in `index.html`:

- **Header Section**: Update name, roll number, institution, and department
- **Education**: Modify degree details, institutions, and dates
- **About Me**: Update the personal description
- **Projects**: Add or modify project details
- **Skills**: Update technical skills and categories
- **Achievements**: Modify achievements and awards
- **Contact**: Update contact information

### Styling
Modify `styles.css` to change:

- **Colors**: Update the color scheme by changing CSS variables
- **Fonts**: Change font families and sizes
- **Layout**: Adjust spacing, margins, and padding
- **Responsive Design**: Modify breakpoints for different screen sizes

## Color Scheme (Dark Theme)

- **Background**: #121212 (Body), #1e1e1e (Container)
- **Primary Blue**: #58a6ff (Headers, titles, accents)
- **Dark Gray**: #2d3748 (Section backgrounds)
- **Success Green**: #48bb78 (Projects, CGPA)
- **Warning Orange**: #ed8936 (Skills)
- **Danger Red**: #f56565 (Achievements)
- **Info Teal**: #38b2ac (Contact)
- **Text Colors**: #e0e0e0 (Primary), #e2e8f0 (Secondary), #a0aec0 (Muted)
